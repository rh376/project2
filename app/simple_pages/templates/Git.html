Git IS Getting

A free and open source containing distributed code management and Version Control System that is accessible under the GNU General Public License version 2. In addition to software version control, Git is used for other applications including configuration management and content management.
Git comes with a variety of features, some of which are listed below. It can help developers utilize code online, update their own code, and more.



  • git add - adds edited files
  • git commit - saves edited files to branch
  • git push - uploads commited branch
  • git pull - retreives repositories from web
  • git merge - combines branches on repositories
  • git clone - copies an existing repository from web
  • git init - turns directory into repository
  • git checkout - switches branches
  • git status - returns state of repository
  • git remote - connects local and remote repositories
  • git branch - determines current branch
  • git stash - save changes without committing
  • git rm - removes files or directories
  • git log - shows chronological order of commits
