Pylint

Pylint is a source-code, bug and quality checker for the Python programming language. It is a tool that checks for errors in Python code, tries to enforce a coding standard and looks for code smells. It can also looks for certain types of errors and recommends suggestions about how blocks can be refactored.

Definitions:



Encapsulation: The process of wrapping up variables and methods into a single entity.
Inheritance: Allows users to define a class that inherits all the methods and properties from another class.
Polymorphism: Lets users define methods in the child class that have the same name as the methods in the parent class.
Abstraction: The process of hiding the real implementation of an application from the user and emphasizing only on how to use the application.
