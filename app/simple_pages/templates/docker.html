Docker

An open source containerization platform. It enables developers to package applications into containers, otherwise known as standardized executable components. They combine application source code with the operating system (OS) libraries and dependencies required to run that code in any environment. Containers simplify delivery of distributed applications, and have become increasingly popular as organizations shift to cloud-native development and hybrid multi-cloud environments.
